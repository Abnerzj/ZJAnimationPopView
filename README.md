# ZJAnimationPopView
ZJAnimationPopView是一个轻量级、无侵入性、可扩展的动画弹框库，支持手写和xib控件，支持横竖屏。动画样式分为显示时和移除时两种类型，两两组合可以配置出72种(8*9)不同的弹框动画效果。
